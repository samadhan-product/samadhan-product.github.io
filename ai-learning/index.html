About this book

AI Learning is the core book in this portal. It is written for people who need to understand how modern AI systems work — without becoming ML researchers.

Each module covers one domain (Transformers, RAG, agents, etc.). Each chapter goes deep on one concept. You can read sequentially or jump to what you need — but the recommended path is designed to compound understanding.
